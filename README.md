# FAENVIPE: Control de Calidad Inteligente en Envases de Vidrio

Este proyecto implementa una **Red Neuronal Artificial** para la detección automática de defectos (Burbujas, Grietas y Manchas) en una línea de producción industrial.

## Objetivo
Reducir la merma en un 15% y mejorar la seguridad estructural mediante la detección de microfisuras con un modelo balanceado de Deep Learning.

## Estructura del Proyecto
- **notebooks/**: Proceso completo de análisis y modelado.
- **results/**: Modelos entrenados (`.keras`) y preprocesadores (`.pkl`).
- **figures/**: Visualización de métricas y análisis de importancia (SHAP).
- **report/**: Documentación técnica y estratégica del proyecto.

## Tecnologías
- Python (Keras, TensorFlow, Scikit-Learn)
- SHAP (Interpretabilidad)
- Pandas & Seaborn
